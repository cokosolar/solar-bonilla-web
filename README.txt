# Solar Bonilla Abogados - Sitio web

Proyecto React/Vite listo para publicar en Vercel.

## Publicar en Vercel

1. Entrar a https://vercel.com
2. Crear cuenta o iniciar sesión.
3. Presionar "Add New..." > "Project".
4. Subir este proyecto o conectarlo desde GitHub.
5. Presionar "Deploy".
6. Luego ir a "Settings" > "Domains" y agregar:
   - solarbonilla.cl
   - www.solarbonilla.cl

## Correcciones necesarias antes de publicar

En `src/main.jsx` reemplazar:
- contacto@solarbonilla.cl
- +56 9 XXXX XXXX

por los datos definitivos.

## Desarrollo local opcional

Si alguna vez quiere probarlo en computador:
- npm install
- npm run dev
